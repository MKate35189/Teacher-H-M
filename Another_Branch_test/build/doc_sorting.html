Сортировка документов

 Содержание этой статьи может немного расходиться с текущей реализацией Riverdoc. Если возникнут вопросы, пожалуйста, обращайтесь по электронной почте guides@ntik.ru.

Сортировка позволяет упорядочить документы по одному из параметров: названию, дате, фамилии владельца. По умолчанию документы отсортированы по дате создания от новых к старым. Параметр и направление сортировки указывается с помощью стрелки вида ↑ или ↓ справа от названия столбца, по которому отсортированы объекты. 

Порядок сортировки

Предусмотрена сортировка по возрастанию и убыванию для каждого из параметров: 

Параметр и направление сортировки

Описание

Имя ↑

Сортировать объекты по названию в следующем порядке: сначала папки, затем документы в направлении 0 → 9, A → Z, А → Я.

Имя ↓

Сортировать объекты по названию в следующем порядке: сначала папки, затем документы в направлении Я → А, Z → A, 9 → 0.

Владелец ↑

Сортировать объекты по фамилии владельца в следующем порядке: сначала папки, затем документы в направлении А → Я.

Владелец ↓

Сортировать объекты по фамилии владельца в следующем порядке: сначала папки, затем документы в направлении Я → А.

Дата создания ↑

Сортировать объекты по дате создания в следующем порядке: сначала папки, затем документы от старых к новым.

Дата создания ↓

Сортировать объекты по дате создания в следующем порядке: сначала папки, затем документы от новых к старым.

Примечание. При переключении между разделами устанавливается порядок сортировки по умолчанию, то есть для документов по дате создания от новых к старым.
