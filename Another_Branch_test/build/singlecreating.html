Как создать документ. Пошаговая инструкция

Важно. На папку, в которой планируете создать документ, должно быть право доступа как минимум "Перезапись версий" (Write).

Чтобы создать документ:

1.Запустите создание одним из способов:

a.Из любого раздела нажмите , далее прикрепите файл с компьютера;

b.Из любого раздела нажмите → Создать документ;

c.перетащите файл с компьютера в подраздел Документы (в любую видимую папку, в которой хотите создать документ).

Система проверит ваши права доступа на папку назначения и, если прав хватает, откроется окно выбора вида документа. Если прав не хватает, система выдаст ошибку.

2.Выберите необходимый вид документа.
В результате откроется окно Создание документа.

3.В области добавления файла прикрепите файл, если нужно:

Область добавления файла

Область добавления файла при наведении курсора

4.Выберите папку назначения или проверьте папку, которую указал Riverdoc.
При необходимости переопределите папку. Для этого нажмите значок справа от названия папки, далее выберите необходимую папку в дереве архива или воспользуйтесь поиском по папкам. Чтобы просмотреть свойства папки назначения, нажмите значок справа.

5.Заполните необходимые поля документа.

6.Сохраните документ с помощью одной из кнопок:

•Создать и открыть – сохранить документ и сразу же открыть его карточку;

•Создать – сохранить документ в системе.

Система проверит, все ли обязательные поля заполнены и, если проверка прошла успешно, созданный документ появится в указанной директории, а анимация подскажет, какой документ был создан. Если в результате проверки оказалось, что какое-либо из обязательных полей не заполнено, система позиционируется на таком поле и выдаёт соответствующую подсказку.
