Версии электронного документа

 Содержание этой статьи может немного расходиться с текущей реализацией Riverdoc. Если возникнут вопросы, пожалуйста, обращайтесь по электронной почте guides@ntik.ru.

Версия электронного документа – сохранённое состояние документа. Документ может иметь одну или несколько версий. Версии одн,ого и того же документа могут отличаться как содержанием файла, так и атрибутами. Также файл может отсутствовать в первых версиях документа до того момента, пока файл не добавят, в последующих версиях файл удалить невозможно (можно только заменить).

Правила создания новой версии/сохранения изменений в текущей версии

При добавлении или изменении файла будет создана новая версия документа.

При редактировании атрибутов подписанного документа изменения сохраняются в новой версии.

При редактировании атрибутов неподписанного документа изменения сохраняются в текущей версии.

Как загрузить новый файл

Загрузить новый можно перетаскиванием файла с компьютера или с помощью элементов интерфейса. В первом случае достаточно перетащить файл с компьютера в карточку документа, далее следовать интерфейсным подсказкам. Обратите внимание, что в документ можно загрузить только один основной файл. При попытке перетащить несколько файлов система оставит один файл, остальные не попадут даже в дополнительные файлы.

Чтобы добавить файл или загрузить новый с помощью элементов интерфейса, следуйте инструкции:

1.В карточке документа нажмите кнопку .

2.В выпадающем списке выберите Загрузить новую версию файла документа.

3.В открывшемся окне выберите и прикрепите нужный файл с компьютера.
В результате будет создана новая версия документа и новый файл будет прикреплён к этой версии. При необходимости пользователь может перейти на вкладку Версии и убедиться, что система загрузила файл.
 

4.Закройте карточку документа.
Для этого нажмите в правом верхнем углу карточки или значок "←" в строке навигации браузера (Alt + ← с клавиатуры). Система вернётся на ту страницу, с которой пользователь открыл карточку документа: список документов или карточка задачи, к которой документ прикреплён.

Важно. Если планируете скачать файл документа, отредактировать у себя на компьютере, а затем загрузить новую версию файла, рекомендуем на время редактирования заблокировать документ (Редактировать → Заблокировать документ). Это поможет избежать конфликта версий, если другой пользователь решит отредактировать документ в это же время. После загрузки новой версии файла не забудьте разблокировать документ (Редактировать → Снять блокировку).

Просмотр версий документа

Все версии документа отображаются на вкладке Версии карточки документа, в которой пользователь может посмотреть количество версий документа, автора и дату создания каждой версии.

Карточка документа. Версии документа

Информация о версиях выводится в виде таблицы и содержит следующую информацию:

•Версия – номер версии и экземпляр документа.

Номер версии – порядковый номер версии документа в формате "Версия №", номер последней версии записывается как "Текущая".
Синим шрифтом и круглой синей меткой выделяется версия документа, карточка которого сейчас открыта. Если открыта карточка последней версии, то номер версии выглядит следующим образом: .

Экземпляр документа – сохранённое состояние документа, содержит карточку версии документа и файл документа. Карточку любой версии можно открыть, просмотреть информацию о документе и получить доступ к версии файла.
Слева от названия версии документа отображается пиктограмма файла, например:

– файл формата doc, docx;

– файл формата xls, xlsx;

– файл формата PDF;

– документ не содержит файла;

– документ содержит файл, но Riverdoc не поддерживает отображение пиктограмм данного формата.

•Дата версии – дата и время создания версии документа.

•Автор версии – сотрудник, который создал текущую версию документа.
Если атрибуты неподписанного документа изменил и сохранил другой сотрудник, то в этом случае изменения сохранятся в текущей версии документа, а автором версии останется сотрудник, который создал версию документа.

Примечание. Количество версий документа не ограничено.

Неактуальная версия документа

При просмотре не последней версии документа Riverdoc выводит информационное сообщение в верхней части страницы:

Нажмите ссылку Перейти к актуальной версии, чтобы открыть последнюю версию документа.

Действия с версиями документа

В любой версии документа можно открыть карточку версии, ознакомиться с атрибутами версии, отобразить предварительный просмотр файла или скачать файл текущей версии. Чтобы открыть карточку версии или скачать файл версии, наведите курсор мыши на нужную версию. Отобразится виджет версии:

В виджете доступны действия Скачать файл и Открыть карточку:

– скачать файл документа текущей версии. Также можно скачать файл документа по щелчку ЛКМ на названии версии.
Если файл отсутствует в документе, то при попытке скачать файл система выведет уведомление "У документа нет файла".

– открыть карточку версии документа.
