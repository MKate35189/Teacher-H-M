Редактирование документа

Основные правила редактирования документа

▪В электронном документе можно отредактировать атрибуты карточки документа, а также добавить файл или загрузить новый.

▪Для редактирования атрибутов блокировка не обязательна.

▪При добавлении или загрузке нового файла документ обязательно блокируется. Возможна либо явная блокировка – по команде пользователя, либо неявная – блокирует система на непродолжительное время, необходимое для сохранения изменений.

▪Минимальное право доступа для блокировки и редактирования документа – "Создание версий" (Version).

▪При сохранении изменений в подписанном документе будет создана новая версия документа, электронные подписи останутся в предыдущей версии. Подписи нужно будет получить заново.

▪Невозможно отредактировать документ, заблокированный другим пользователем.

▪Если до начала редактирования документ был заблокирован, то после сохранения изменений блокировка остаётся.

Правила создания новой версии/сохранения изменений в текущей версии

При добавлении или изменении файла будет создана новая версия документа.

При редактировании атрибутов подписанного документа изменения сохраняются в новой версии.

При редактировании атрибутов неподписанного документа изменения сохраняются в текущей версии.

Как отредактировать атрибуты документа

Чтобы отредактировать атрибуты документа:

1.В карточке документа нажмите кнопку .

2.В выпадающем списке выберите Редактировать атрибуты....

3.В открывшемся окне измените значение нужных атрибутов.

4.Нажмите Сохранить.
В результате окно Редактирование атрибутов закроется, система вернётся в карточку документа.

Чтобы закрыть карточку документа, нажмите в правом верхнем углу карточки или значок "←" в строке навигации браузера (Alt + ← с клавиатуры). Система вернётся на ту страницу, с которой пользователь открыл карточку документа: список документов или карточка задачи.

Как добавить файл или загрузить новый

Добавить файл или загрузить новый можно перетаскиванием файла с компьютера или с помощью кнопок. В первом случае достаточно перетащить файл с компьютера в карточку документа, далее следовать интерфейсным подсказкам. Обратите внимание, что в документ можно загрузить только один основной файл. При попытке перетащить несколько файлов система оставит один файл, остальные не попадут даже в дополнительные файлы.

Чтобы добавить файл или загрузить новый с помощью элементов интерфейса, следуйте инструкции:

1.В карточке документа нажмите кнопку .

2.В выпадающем списке выберите Загрузить новую версию файла документа.

3.В открывшемся окне выберите и прикрепите нужный файл с компьютера.
В результате будет создана новая версия документа и новый файл будет прикреплён к этой версии. При необходимости пользователь может перейти на вкладку Версии и убедиться, что система загрузила файл.
 

4.Закройте карточку документа.
Для этого нажмите в правом верхнем углу карточки или значок "←" в строке навигации браузера (Alt + ← с клавиатуры). Система вернётся на ту страницу, с которой пользователь открыл карточку документа: список документов или карточка задачи, к которой документ прикреплён.

Важно. Если планируете скачать файл документа, отредактировать у себя на компьютере, а затем загрузить новую версию файла, рекомендуем на время редактирования заблокировать документ (Редактировать → Заблокировать документ). Это поможет избежать конфликта версий, если другой пользователь решит отредактировать документ в это же время. После загрузки новой версии файла не забудьте разблокировать документ (Редактировать → Снять блокировку).

Примечание. Список доступных действий в меню зависит от нескольких параметров:

•прав пользователя на документ,

•заблокирован ли документ,

•подписан ли документ,

•есть ли файл в текущей версии документа.
