Редактирование файла документа

 Содержание этой статьи может немного расходиться с текущей реализацией Riverdoc. Если возникнут вопросы, пожалуйста, обращайтесь по электронной почте guides@ntik.ru.

Правила при редактировании файла

▪При редактировании или удалении файла всегда создаётся новая версия документа.

▪При сохранении изменений в файле документ обязательно блокируется. Возможна либо явная блокировка – по команде пользователя, либо неявная – блокирует система на непродолжительное время, необходимое для сохранения изменений.

Как загрузить новую версию файла

Чтобы загрузить новую версию файла:

1.В карточке документа выберите команду Редактировать → Загрузить новую версию файла документа.

2.В открывшемся окне выберите и прикрепите нужный файл с компьютера.
В результате будет создана новая версия документа и новый файл будет прикреплён к этой версии. При необходимости пользователь может проверить все версии текущего документа.

3.Закройте карточку документа.
Для этого нажмите в правом верхнем углу карточки или значок "←" в строке навигации браузера (Alt + ← с клавиатуры). Система вернётся на ту страницу, с которой пользователь открыл карточку документа: список документов или карточка задачи, к которой документ прикреплён.
