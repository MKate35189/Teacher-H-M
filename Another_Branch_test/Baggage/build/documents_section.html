Интерфейс раздела "Документы"

Для удобства документы в разделе сгруппированы в подразделы: Недавние, Избранное, Документы. Содержимое выбранного подраздела отображается справа. По умолчанию открывается содержимое подраздела Недавние.

Раздел "Документы"

Области:

[1] Перейти к стартовой странице системы: раздел Задачи → Делаю (линейный список).

[2] Выбрать раздел.

[3] Открыть справку | Перейти в службу поддержки.

[4] Открыть профиль пользователя | Выйти из системы.

[5] Загрузить файлы с компьютера.

[6] Создать документ | Запустить процесс | Создать поручение | Создать папку.

[7] "Область поиска документов".

[8] "Меню документов".

[9] "Строка навигации".

[10] "Заголовок таблицы".

Важно. В столбце Дата создания для папки выводится дата создания папки, для документа – дата создания последней версии документа.

[11] "Список документов и папок".

[12] "Область представлений".

[13] "Постраничная навигация".

Навигация в разделе "Документы"

Перемещение по разделу возможно как в дереве документов [6], так и в области представлений [10].

В дереве документов:

•одинарный щелчок по названию подраздела выводит его содержимое в области представлений [10];

•одинарный щелчок по названию папки выводит список документов и вложенных папок в области представлений [10];

•одинарный щелчок по стрелке / разворачивает содержимое подраздела или папки в дереве документов [6] и отображает вложенные папки, при этом список документов в области представлений [10] не меняется.

•одинарный щелчок по стрелке / сворачивает содержимое подраздела или папки, при этом список документов в области представлений [10] не меняется.

В области представлений:

•в списке документов по папкам;

•с помощью строки навигации:

Строка навигации

Чтобы перейти в папку, нажмите ЛКМ на названии нужной папки.

Примечание. В дереве документов текущий выбранный подраздел или папка выделяется синим цветом.

Как открыть карточку документа

Карточку документа можно открыть как в списке документов, так и из карточки задачи, к которой этот документ прикреплён. Для этого нужно нажать значок в виджете документа, который появляется при наведении курсора мыши на документ.

Как открыть карточку документа

При одинарном нажатии ЛКМ на названии документа Riverdoc скачивает файл документа (если не настроен запрет на уровне операционной системы или браузера) либо выводит сообщение о том, что файл отсутствует в документе.
