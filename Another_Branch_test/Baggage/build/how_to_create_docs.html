Пошаговые инструкции по созданию документов

Примечание. Минимальное право доступа для создания документа или папки – "Создание версий" на директорию, в которой пользователь планирует создать объект.

Как создать документ

Чтобы создать документ в Riverdoc:

1.Нажмите + Создать → Создать документ. Откроется окно Создание документа.

2.Укажите подходящий вид документа.

3.При необходимости прикрепите файл в области добавления файла:

Область добавления файла

Область добавления файла при наведении курсора

4.Выбор папка назначения:

a.При создании документа из раздела Документы проверьте, правильно ли указана папка назначения.

Если нужно, переопределите папку. Для этого нажмите значок справа от названия папки, далее выберите необходимую папку в дереве архива или воспользуйтесь поиском по папкам. Чтобы просмотреть свойства текущей папки назначения, нажмите значок справа.

b.При создании документа не из раздела Документы папку назначения определяет Riverdoc в зависимости от указанного вида документа. Если папка назначения для текущего вида не задана, укажите нужную папку с помощью кнопки Выбрать папку.

5.Заполните необходимые поля документа.

6.Сохраните документ с помощью одной из кнопок:

•Создать и открыть – сохранить документ и сразу же открыть его карточку;

•Создать – сохранить документ в системе.

В результате созданный документ появится в указанной директории.
Как создать документы одного вида

Чтобы создать несколько документов за один раз:

1.Нажмите кнопку Загрузить файлы.

2.В открывшемся окне выберите с компьютера файлы, для которых хотите создать документы в СЭД, нажмите Открыть. В результате форма создания документов обновится, образы файлов с указанием их количества добавятся в окне Создание документов.

3.Определитесь, документы каких видов будете создавать:

a.Если планируете создать документы одного вида, в справочнике ниже выберите подходящий вид для всех документов.

b.Если планируете создать документы разных видов, нажмите ссылку Это документы разных видов.

4.Далее последовательно для каждого документа:

a.Проверьте, правильно ли указан вид документов (если создаёте документы одного вида).

b.Укажите подходящий вид для каждого документа (если создаёте документы разных видов).

5.Проверьте, правильно ли указана папка назначения.

Если нужно, переопределите папку. Для этого нажмите значок справа от названия папки, далее выберите необходимую папку в дереве архива или воспользуйтесь поиском по папкам. Чтобы просмотреть свойства текущей папки назначения, нажмите значок справа.

6.Заполните необходимые поля документа.

7.Нажмите Создать.

В результате созданные документы появятся в папке(ах) назначения.

Примечание. Если пользователь прерывает массовое создание документов, то уже созданные документы сохраняются в системе, а создание документов из оставшихся файлов отменяется. Riverdoc выведет соответствующее предупреждение, от пользователя потребуется подтверждение отмены.
Как создать документы разных видов

Чтобы создать несколько документов с помощью перетаскивания:

1.Выберите на рабочем столе или в файловом менеджере файлы, для которых хотите создать документы в СЭД, и с помощью мыши переместите их в папку назначения. При перетаскивании создание документов можно инициировать:

oв текущей папке;

oв любой папке, вложенную в текущую;

oв дереве документов (в любой "развёрнутой " папке).

В результате откроется окне Создание документов, образы файлов с указанием их количества добавятся в форме создания документов.

2.Определитесь, документы каких видов будете создавать:

a.Если планируете создать документы одного вида, в справочнике ниже выберите подходящий вид для всех документов.

b.Если планируете создать документы разных видов, нажмите ссылку Это документы разных видов.

3.Далее последовательно для каждого документа:

a.Проверьте, правильно ли указан вид документов (если создаёте документы одного вида).

b.Укажите подходящий вид для каждого документа (если создаёте документы разных видов).

4.Проверьте, правильно ли указана папка назначения.

Если нужно, переопределите папку. Для этого нажмите значок справа от названия папки, далее выберите нужную папку в дереве архива или воспользуйтесь поиском по папкам. Чтобы просмотреть свойства папки назначения, нажмите значок справа.

5.Заполните необходимые поля документа.

6.Нажмите Создать.

Примечание. Если пользователь прерывает массовое создание документов, то уже созданные документы сохраняются в системе, а создание документов из оставшихся файлов отменяется. Riverdoc выведет соответствующее предупреждение, от пользователя потребуется подтверждение отмены.
