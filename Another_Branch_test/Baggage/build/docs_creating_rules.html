Способы и правила создания документов

Основные правила создания документов в Riverdoc

▪В корне раздела Документы объекты может создавать только администратор из Консоли администрирования.

▪Минимальное право доступа для создания документа или папки – "Создание версий" на директорию, в которой пользователь планирует создать объект.

▪В Riverdoc создать документ можно с помощью элементов интерфейса (кнопок) или с помощью технологии drag-and-drop.

▪При создании объекта из раздела Документы в качестве папки назначения автоматически указывается текущая папка. Папку назначения можно изменить в процессе создания.

▪Документ можно создать из любого раздела с помощью кнопки + Создать. Папку назначения определяет Riverdoc в зависимости от указанного вида документа. Если папка назначения для текущего вида не задана, необходимо указать папку вручную.

▪При создании документа в Riverdoc необходимо указать вид документа. Изменить вид документа можно только до первого сохранения карточки документа.

▪Если в процессе создания документа пользователь прикрепляет файл, то название файла используется в качестве названия нового документа. Название можно изменить в любой момент.

▪Riverdoc позволяет загружать файлы любых форматов, ограничение размера загружаемого файла устанавливает администратор системы. Количество файлов, загружаемых за один раз, не ограничено.

Способы создания документов в Riverdoc

В Riverdoc документ(ы) можно создать одним из следующих способов:

Способ создания объекта

Описание способа

Загрузить несколько файлов с компьютера. На основании этих файлов в Riverdoc будут созданы документы одного или разных видов.

(в разделе Документы)

Создать папку или документ (только один документ за один раз). В качестве папки назначения автоматически указывается текущая папка.

(в правом верхнем углу)

Создать документ (только один документ за один раз). Папку назначения определяет Riverdoc в зависимости от указанного вида документа. Если папка назначения для текущего вида не задана, необходимо указать папку вручную.

Перетаскивание (drag-and-drop)

Создать документ(ы) путём перетаскивания файла(ов) с рабочего стола или из файлового менеджера. При перетаскивании документ(ы) можно создать в текущей папке, в любой папке, вложенную в текущую, а также в дереве документов (в любой "развёрнутой " папке).
