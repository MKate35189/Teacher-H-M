Заполнение параметров поручения

В окне Создать поручение постановщику необходимо указать параметры поручения (инструкцию, исполнителя, срок исполнения). Ниже приведён скриншот окна создания поручения с размеченными и пронумерованными областями, а также пояснениями по каждой области.

Окно создания поручения

[1] Важно (флажок) – метка важности, используется в дальнейшем для сортировки и упорядочивания важных задач;

[2] – прервать создание поручения и закрыть окно.

[3] Инструкция* – краткая инструкция к поручению, обязательна для заполнения.
В дальнейшем добавляется в название задачи по поручению и отображается в списке задач, в карточке задачи. Ограничение по количеству символов – 250.

[4] Срок исполнения (выпадающий календарь) – крайний срок выполнения поручения. Это поле обязательно для поручений с контролем.
Срок исполнения можно указать с помощью цифр на клавиатуре, выбрать день в календаре или воспользоваться автозаполнением даты: сегодня, 1 день, 2 дня, 3 дня, неделя, 2 недели, месяц.

[5] Исполнители* – Фамилия И. О. сотрудника или сотрудников, кому поступит задача на исполнение.
Начните вводить фамилию, а затем выберите фамилию нужного сотрудника из предлоденного списка. Ограничений по количеству исполнителей нет.

[6] Расширенный поиск / Расширенный поиск (ссылка) – открывает окно Выбор исполнителей, в котором можно выбрать всех сотрудников структурного подразделения за один раз или выбрать сотрудника в структуре организации (например, если постановщик не знает или забыл фамилию сотрудника, работающего в другом филиале).

[7] Документ | Прикрепить (ссылка) – открывает окно Добавление документа, в котором пользователь может выбрать и прикрепить документ одним из следующих способов:

•Поиск (поле) – выбрать документ из результатов поиска в электронном архиве.

•Выбрать из архива (ссылка) – выбрать документ в дереве электронного архива.

•Выбрать документ из числа последних десяти документов.

К карточке задачи по поручению можно прикрепить только один документа. После запуска поручения удалить или изменить документ невозможно.

       Как добавить/удалить документ

После того как нужный документ будет найден, нажмите кнопку Выбрать справа от названия документа. В результате название прикреплённого документа отобразится под областью Документ | Прикрепить. Слева от названия документа отображается пиктограмма файла документа (например, / / ). Если документ не содержит файл, тогда отображается пиктограмма . Если файл есть, но Riverdoc не поддерживает отображение пиктограмм данного формата, отображается .

В случае ошибочного выбора документ можно открепить от карточки поручения с помощью значка или выбрать другой документ.
Также можно скачать файл документа на компьютер, для этого достаточно щёлкнуть по названию документа один раз. При попытке скачать документ без файла система выдаст предупреждение "У документа нет файла".

[8] Подробности – подробное описание сути поручения в виде текстового и/или голосового* сообщения.
Ограничение на количество символов в поле – 1500. Для текстового сообщения доступны функции форматирования текста: различное начертание шрифта (полужирный, курсив, подчёркнутый), а также маркированный и нумерованный список.

* Ввод голосового сообщения поддерживается только в браузере Chrome. Для записи голосового сообщения на компьютере пользователя должен быть подключен и настроен микрофон.

[9] Контролёр – сотрудник, который будет проверять и согласовывать поручение;

[10] Я контролёр (кнопка) – нажмите эту кнопку, чтобы указать себя в качестве контролёра по поручению;

[11] Автор поручения* – сотрудник, который будет числиться автором задачи по поручению. По умолчанию автором поручения указывается постановщик. Автора можно переопределить до запуска поручения. В некоторых случаях в качестве автора постановщика нужно указать другого сотрудника (не постановщика). Например, постановщиком поручения является секретарь руководителя, но в качестве автора поручения необходимо указать именно руководителя.

[12] Я автор (кнопка) – нажмите эту кнопку, чтобы указать себя в качестве автора поручения;

[13] Кнопки действий:

•Создать поручение – по нажатию этой кнопки система проверит, все ли обязательные поля заполнены и, если проверка прошла успешно, поручение поступит исполнителям. Если в результате проверки оказалось, что какое-либо из обязательных полей не заполнено или заполнено некорректно, система позиционируется на таком поле и выдаёт соответствующую подсказку.

•Отмена – отменить создание поручения и закрыть окно.

После успешного создания поручения задачи поступят исполнителям в течение нескольких секунд.
