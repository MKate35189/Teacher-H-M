Системная фильтрация

Фильтр по группам задач (меню задач) – системный фильтр, содержит преднастроенный список разделов задач и поручений. При выборе определённого раздела/подраздела отображаются задачи нужной группы. Например, Делаю → Срочные или Контролирую → Просроченные. 

По умолчанию в меню задач содержатся следующие группы:

•Делаю

oСрочные

oПросроченные

•Поручил

oПросроченные

•Контролирую

oСрочные

oПросроченные

•Избранные

Пользователь может скрыть группы задач, которые он не использует. Новые группы задач может добавить только администратор и только в раздел Делаю. 

Настройка системного фильтра

Чтобы настроить список групп, отображаемых в меню задач, нажмите . В открывшемся окне на вкладке Отображение фильтров перечислены все доступные группы и подгруппы задач. Установите флажок напротив тех групп/подгрупп, которые необходимо выводить в меню задач. Сохраните настройки.

На вкладке Срочные задачи (поручения) пользователь может изменить срок, при наступлении которого задача или поручение становится срочной(ым). Единицы измерения: дни, недели, месяцы. По умолчанию установлено значение три дня.

Делаю

Искать по:

•Задаче – поиск по названию задачи (например, "Исполнение.", "Подписать.").

•Инструкции – поиск по инструкции к задаче/поручению. В общем списке задач Делаю инструкция следует после названия задачи.

•Отправителю – поиск по отправителю задачи/поручения.

•Имени процесса/поручения – поиск по названию процесса или поручения.
Поручил

Искать по:

•Инструкции – поиск по тексту инструкции

•Контролёру – поиск по куратору поручения.

•Имени поручения – поиск по названию поручения.
Контролирую

Искать по:

•Инструкции – поиск по тексту инструкции

•Автору поручения – поиск по автору поручения.

•Имени поручения – поиск по названию поручения.
Избранные

"Вертикальная" фильтрация по группам Делаю/Поручил/Контролирую.
