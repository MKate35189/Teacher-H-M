Запуск процесса

В Riverdoc запустить процесс можно из любого раздела с помощью кнопки + Создать, также можно запустить процесс по конкретному документу из карточки этого документа.

Запуск процесса из раздела "Задачи"

Чтобы запустить процесс из раздела Задачи:

1. Нажмите Создать → Запустить процесс.

Запуск процесса из раздела "Задачи"

Откроется окно Запуск процесса, в котором содержатся все процессы документооборота компании.

2. Укажите нужный процесс. Для этого начните вводить название процесса в поле Поиск или выберите из предложенного списка.

3. Далее наведите курсор мыши на название нужного процесса и нажмите Выбрать.

Выбор процесса

В результате откроется окно запуска процесса, в котором необходимо указать параметры процесса.

Запуск процесса из раздела "Документы"

Чтобы запустить процесс из раздела Документы:

3. Далее наведите курсор мыши на название нужного процесса и нажмите Выбрать.

Выбор процесса

В результате откроется окно запуска процесса, в котором необходимо указать параметры процесса.

Запуск процесса из карточки документа

Чтобы запустить процесс из раздела Документы:

1. Откройте карточку документа, по которому хотите запустить процесс.

2. В карточке документа нажмите кнопку Запустить процесс.

Запуск процесса из карточки документа

В результате откроется окно Запуск процесса, в котором содержатся процессы, релевантные для документа текущего вида.

3. Укажите нужный процесс. Для этого начните вводить название процесса в поле Поиск или выберите из предложенного списка.

4. Далее наведите курсор мыши на название нужного процесса и нажмите Выбрать.

Выбор процесса

В результате откроется окно запуска процесса, в котором необходимо указать параметры процесса.
