Дополнительные файлы и связанные документы

Пользователь может прикрепить к карточке документа дополнительные файлы и связанные документы. Например после подписания договора можно прикрепить подписанную скан-копию, затем связать договор со счетами на оплату и актом выполненных работ. Таким образом, предоставить доступ к полному пакету документов из карточки основного ("родительского") документа.

Карточка документа с дополнительным файлом и свзанными документами

Дополнительные файлы

Дополнительные файлы – файлы, прикреплённые к карточке документа с компьютера пользователя.

Чтобы прикрепить файл с ПК к карточке документа:

1.Нажмите ссылку + Загрузить справа от Дополнительные файлы.

2.В открывшемся окне выберите файл, подтвердите выбор.

В результате система добавит файл к карточке документа, счётчик дополнительных файлов увеличится на единицу. Если нужно прикрепить несколько файлов, повторите действия 1-2 для каждого загружаемого файла. Прикрепить несколько файлов за один раз невозможно.

Связанные документы

Связанные документы – два документа электронного архива, между которыми пользователь установил связь.

Чтобы связать два документа электронного архива:

1.Нажмите ссылку + Добавить справа от Связанные документы.

2.В открывшемся окне выберите один или несколько документов.
Выбрать документ можно:

oиз числа последних документов;

oвоспользовавшись поиском;

oв дереве электронного архива (ссылка Выбрать из архива).

3.С помощью кнопки Выбрать эти документы подтвердите выбор.

В результате система добавит связь между текущим документом и указанным(и) документом(ами), счётчик связанных документов увеличится на число добавленных связей.

Примечание. Связь документов отражается в карточках связанных между собой документов, а также в карточках задач, по которым документы проходят.
