Недавние

 Содержание этой статьи может немного расходиться с текущей реализацией Riverdoc. Если возникнут вопросы, пожалуйста, обращайтесь по электронной почте guides@ntik.ru.

Недавние – список документов, с которыми текущий пользователь работал в последнее время.
Содержимое этого подраздела отображается по умолчании при загрузке раздела Документы.

"Недавние". Основные правила

▪При первом входе в систему и до первого взаимодействия пользователя с документами подраздел будет пустым.

▪В подраздел попадают документы, с которыми взаимодействовал только текущий пользователь. Последующие взаимодействия других пользователей с этими же документами не учитываются в данном контексте.

▪В этом подразделе выводятся документы, которые текущий пользователь создал, просмотрел или изменил.

▪Если текущий пользователь выполнил с одним и тем же документом несколько операций (например, создал и изменил атрибуты), то в Недавние отображается информация по последнему событию.

▪Документы в Недавние выводятся в виде блоков по дням. Блоки сортируются по дате взаимодействия пользователя с документом: от последних к более ранним.

▪При скачивании документа или использовании шаблона документа добавляется запись о просмотре документа, даже если пользователь не открывал карточку документа.

▪В Недавние отображается до 50-ти документов, с которыми пользователь работал в последнее время.

▪Сократить количество отображаемых документов можно с помощью флажков:

oСозданные и изменённые – при включенном флажке выводятся документы, которые пользователь создал и/или изменил.

oПросмотренные – при включенном флажке выводятся документы, которые пользователь просмотрел.
